# BDD using NodeJs and cucumberJs
https://dev.to/ynmanware/bdd-for-rest-api-using-cucumber-js-2pol

#### install 
```
npm i
npm install -g json-server
```

#### start mock REST API
```
json-server --watch db.json
```

#### start test 
```
npm run gaji_gesa
```
